## 🛠️ TROUBLESHOOTING GUIDE

### Common Issues & Solutions

#### Icons Not Generating
```
Issue: "No module named 'PIL'" or "PIL not found"
Solution: pip install Pillow --upgrade
Then: python generate_icons.py
```

#### APK Build Fails
```
Issue: Various compilation errors
Solution: 
  1. flutter clean
  2. flutter pub get
  3. flutter build apk --release -v (for verbose output)
  4. Check flutter doctor for issues
```

#### Windows Build Fails
```
Issue: Visual Studio not found or missing components
Solution:
  1. Install Visual Studio Build Tools
  2. Run flutter doctor and follow suggestions
  3. Restart computer if needed
  4. Try building again
```

#### Web Build Issues
```
Issue: Various errors during build
Solution:
  1. flutter clean
  2. flutter pub get
  3. flutter build web --verbose
  4. Check for dart errors: dart analyze
```

**For more details, see BUILD_SETUP.md**
